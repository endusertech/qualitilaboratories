# Qualiti Laboratories

**Provider of Comprehensive Chemical Programs for High Risk Environments**

## Logo Design Project (GLA-11)

This repository contains the logo design assets for Qualiti Laboratories.

### Project Brief

Create a professional logo for "Qualiti Laboratories" including the tagline: "Provider of Comprehensive Chemical Programs for High Risk Environments"

### Directory Structure

- `/assets/` - Final logo assets (PNG, SVG, etc.)
- `/src/` - Source design files (e.g., .ai, .fig, .sketch)
- `/exports/` - Various export sizes and formats
- `DESIGN_BRIEF.md` - Detailed design requirements

### Getting Started

1. All design work should be done in the `/src/` directory
2. Final assets should be exported to `/assets/` and `/exports/`
3. Follow the design brief in `DESIGN_BRIEF.md`
